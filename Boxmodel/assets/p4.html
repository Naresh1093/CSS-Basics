CSS BOX MODEL..!

every element has a display on a web page is comprised of one or more rectangular boxes. CSS box model typically describes how these rectangular boxes are laid out a web page. This boxes can have different properties and can intract with each other in different ways, but every box has a content area and optional surrounding padding,border,and margin areas.
